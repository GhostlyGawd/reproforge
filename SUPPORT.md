# Support

ReproForge is a pre-alpha prototype with no service-level agreement or guaranteed response time.

For reproducible bugs, documentation problems, or focused feature proposals, open a [GitHub issue](https://github.com/GhostlyGawd/reproforge/issues). Include:

- the ReproForge commit;
- operating system, Node.js version, and npm version;
- the exact command and complete sanitized output;
- expected and observed behavior;
- whether the trusted control command succeeds; and
- a minimal fixture or bundle when safe to share.

Search existing issues before filing. General environment debugging, production incident response, arbitrary-repository execution, and recovery of sensitive data are outside the current support scope.

Do not post vulnerability details, credentials, private repositories, or personal data in a public issue. Use the private path described in [SECURITY.md](SECURITY.md).
